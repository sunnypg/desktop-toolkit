## 🍉Installation

1. 下载源代码：
```bash
git clone https://github.com/sunnypg/desktop-toolkit.git
```
2. 安装依赖：
```bash
cd desktop-toolkit
npm install
```
3. 启动：
```bash
npm run dev
```
更多设置请查阅：[electron-vite](https://cn.electron-vite.org/)
